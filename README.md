# Ecuaciones-juego

Prototipo educativo "Caída libre"

Descripción:
- Minijuego prototipo hecho en HTML/CSS/JS para ilustrar caída libre y cómo la masa y resistencia afectan la aceleración.
- Tres pelotas (masas 0.2, 1.0 y 5.0 kg).
- Cinco mundos con parámetros distintos (gravedad, arrastre).
- Empiezas a 100 m, velocidad inicial 0. El objetivo es tocar una plataforma o el suelo; tu puntuación es el tiempo que tardas (menor = mejor).

Archivos:
- `index.html` — interfaz y canvas.
- `style.css` — estilos.
- `src/game.js` — lógica del juego.

Cómo probar:
1. Abre `index.html` en un navegador moderno (o sirve con `python -m http.server` y visita http://localhost:8000).
2. Selecciona el mundo y la pelota.
3. Presiona Iniciar. Usa ← y → para moverte horizontalmente.

Notas y siguientes pasos sugeridos:
- Mejorar colisiones y comportamiento de rebote.
- Añadir gráficos, sonidos y niveles más pulidos.
- Añadir una pantalla de puntajes y explicación interactiva de la ecuación de caída libre.

Hecho como prototipo educativo por petición del usuario.
